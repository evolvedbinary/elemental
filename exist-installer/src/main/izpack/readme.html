Launching Elemental

On Windows and Linux, use the Elemental start menu item or desktop icon for launching Elemental. On Mac OS, double-click the Elemental.app icon inside the folder in which Elemental is installed.

You can also launch Elemental by double-clicking on exist-start.jar or by calling "java -jar exist-start.jar" on a command line.

Launching Elemental in one of these ways will display a splash screen while starting up the database. After startup, you will find an Elemental icon in your system tray/menu bar. Click (Windows and Mac OS) or right-click (Linux) on the icon to get a popup menu with further options.
